# Amazon Product Data Analysis
Data analysis of Amazon products using Python, Excel and Power BI
Анализ данных о продуктах Amazon с целью выявления продаж, скидок и доходности категорий.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- Power BI
- Excel

## Data
- `all car and motorbike products.csv` – исходные данные, одна категория и подкатегория объединены в одно поле
- `amazon_car_and_motorbike_clean.xlsx` – очищенные данные для анализа и построения дашбордов

## Analysis
- Очистка данных (удаление пропусков, корректировка типов)
- Построение графиков распределения продаж, цен и скидок
- Генерация Excel с готовыми данными для Power BI

## Dashboard
- Power BI дашборд с ключевыми метриками:
 - Анализ цен и скидок: гистограмма по количеству товара и стоимости
- Рейтинг популярности товаров: точечная диаграмма с размером точки по цене
- ТОП-10 товаров: таблица
- Анализ скидок: столбчатая диаграмма + гистограмма распределения скидок.

## Key Insights
1.Распределение скидок:
  a. Большинство товаров имеют скидку в диапазоне 20–50%;
  b. Слишком высокие скидки (более 90%) встречаются редко и, вероятно, являются аномалиями данных.
2. Рейтинг и популярность товаров:
  a. Товары с высоким рейтингом (4–5 звезд) часто имеют большое количество отзывов, но есть и нишевые товары с высоким рейтингом и малым количеством отзывов;
  b. Соотношение рейтинга и количества отзывов помогает выявлять лидеров продаж и потенциально перспективные товары.
3. ТОП-10 товаров:
  a. Выявлены товары с наибольшим количеством отзывов и высоким рейтингом;
  b. Эти товары представляют интерес для аналитики продаж, закупок и маркетинга.
4. Аналитические выводы:
  a. Можно оптимизировать маркетинговые акции, ориентируясь на популярные категории и подкатегории;
  b. Распределение скидок и цен позволяет планировать акции и прогнозировать спрос;
  c. Интерактивные дашборды позволяют быстро находить отклонения и принимать решения на основе данных.
